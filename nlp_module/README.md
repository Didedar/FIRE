# NLP Module (Placeholder)

Этот модуль предназначен для AI-анализа текста обращений клиентов.  
**Статус: НЕ РЕАЛИЗОВАН** — Backend вызывает URL ниже и при ошибке возвращает мок-данные.

## API Contract

### `POST http://nlp-service:8001/analyze`

**Request:**
```json
{
  "text": "Текст обращения клиента",
  "address": "Казахстан, Алматы, ул. Абая, 10"
}
```

**Response:**
```json
{
  "type": "Жалоба",
  "tonality": "Негативный",
  "priority": 8,
  "language": "RU",
  "summary": "Клиент жалуется на списание средств без уведомления. Рекомендуется связаться и разобраться.",
  "latitude": 43.238949,
  "longitude": 76.945465
}
```

### Возможные значения

| Поле | Значения |
|------|----------|
| `type` | Жалоба, Смена данных, Консультация, Претензия, Неработоспособность приложения, Мошеннические действия, Спам |
| `tonality` | Позитивный, Нейтральный, Негативный |
| `priority` | 1–10 |
| `language` | KZ, ENG, RU |
